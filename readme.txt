Automate generation of leads to Teamgate CRM by creating contacts, subscriptions or other kind of forms within your WordPress website. 

== Description ==

Teamgate CRM forms for WordPress allows you to connect your WordPress website with your [Teamgate](https://www.teamgate.com/) account. Your website visitors will be able to fill up the forms you created. A new Lead, Contact or a new Deal will be created from data entered by the website visitor. By automating Lead generation process, you will never lose your potential customers. This way, you can focus more on working with the information sent to Teamgate CRM.

To embed your forms on your WordPress website, you need a Teamgate account. [Sign in](https://my.teamgate.com/) or [Sign up](https://www.teamgate.com/trial/) for Teamgate.

Teamgate plugin is a Contacts Forms extention, so, in order to install Teamgate plugin for your WordPress website, first you need to make sure you have Contacts Forms plugin installed already. The plugin will generate a short code that need to be put into your WordPress website editor, in order to see the created form in your website.

== Installation ==

1. Install the plug-in from the WordPress plug-in installer or upload `plug-in` to the `/wp-content/plugins/` directory
2. Activate the plug-in through the `Plug-ins` menu in WordPress
3. Enter your Teamgate.com `APP_KEY` from your `Your Teamgate account -> Settings -> Additional Features -> External Apps` and your `AUTH_TOKEN` from `Your Teamgate account -> My Profile -> Preferences`.
